# unit-converter
Simple web app that can convert between different units of measurement.

It can convert units of length, weight, temperature. The user can input a value and select the units to convert from and to. The application will then display the converted value.

Units:
LENGTH_UNITS = ['millimeter', 'centimeter', 'meter', 'kilometer', 'inch', 'foot', 'yard', 'mile']
WEIGHT_UNITS = ['milligram', 'gram', 'kilogram', 'ounce', 'pound']
TEMPERATURE_UNITS = ['Celsius', 'Fahrenheit', 'Kelvin']

Installation

git clone https://github.com/SalikNikodem/unit-converter.git

```bash
python -m venv venv
venv\Scripts\activate

pip install flask

python app.py

Server adress
http://127.0.0.1:5000
```
